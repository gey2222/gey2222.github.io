HaloProbe: Bayesian Detection and Mitigation of Object Hallucinations in Vision-Language Models

HaloProbe：视觉语言模型中对象幻觉的贝叶斯检测与缓解
Reihaneh Zohrabi, Hosein Hasani, Akshita Gupta, Mahdieh Soleymani Baghshah, Anna Rohrbach, Marcus Rohrbach
2026-04-07
arxiv 2604.06165 ↗
TL;DR 揭示 attention-based 幻觉检测的 Simpson's 悖论（token 位置和对象重复是隐藏混淆因子）。HaloProbe 用贝叶斯框架分离内外部信号，作为外部评分引导解码，比侵入式方法更有效且不损流畅度。

📌 研究背景与动机

大型视觉语言模型在图像描述中会产生对象幻觉问题。现有方法普遍用模型对视觉 token 的 attention 权重作为检测信号，但本文揭示这存在根本性缺陷。

核心发现：Simpson's 悖论

  • Token 位置和对象在描述中的重复次数是隐藏混淆因子（hidden confounders）。
  • 聚合统计后，attention 趋势会反转或消失——这正是 Simpson's 悖论的经典表现。
  • 因此，粗粒度 attention 分析作为幻觉检测信号是不可靠的。

⚙️ 核心方法：HaloProbe

贝叶斯分解框架：将 token 级幻觉概率估计分解为外部描述统计（token 位置、对象重复等）和内部解码信号的乘积，分别建模后合并为后验概率。
平衡训练：通过平衡训练隔离内部解码证据，消除外部混淆因子的影响；再结合外部特征学习先验，恢复真实后验。
非侵入式缓解：将 HaloProbe 作为外部评分信号用于解码引导，不修改模型内部参数，避免侵入式方法降低输出质量的问题。

📊 实验结果

  • HaloProbe 引导解码比 SOTA 侵入式干预方法更有效地减少幻觉。
  • 保持输出流畅度和实用性，无质量损失。
