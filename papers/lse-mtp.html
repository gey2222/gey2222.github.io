Toward Consistent World Models with Multi-Token Prediction and Latent Semantic Enhancement (LSE-MTP)

面向一致性世界模型的多 Token 预测与隐语义增强
Qimin Zhong, Hao Liao, Haiming Qin, Mingyang Zhou, Rui Mao, Wei Chen, Naipeng Chao
2026-04-07
ACL 2026 Main Conference
arxiv 2604.06155 ↗
TL;DR 理论证明 MTP 通过梯度耦合推动 LLM 内部信念状态收敛，但标准 MTP 会诱发"结构性幻觉"（隐空间走捷径）。LSE-MTP 将预测锚定到真实隐状态轨迹，消除幻觉、增强鲁棒性。

📌 研究背景与动机

LLM 是否构建了连贯的内部世界模型仍有争议。Multi-Token Prediction (MTP) 相比 Next-Token Prediction (NTP) 展示了更结构化的表示学习潜力，但其机制和局限尚未被系统分析。

核心发现

  • MTP 的归纳偏置：理论分析表明 MTP 通过梯度耦合机制诱导表示收缩性（contractivity），推动内部信念状态收敛，有助于构建一致的世界模型。
  • 结构性幻觉：标准 MTP 的离散 token 监督会鼓励模型在隐空间学习非法捷径（违反环境约束的表示），即"结构性幻觉"。

⚙️ 核心方法：LSE-MTP

Latent Semantic Enhancement：在 MTP 训练中将预测锚定到真实的隐状态轨迹，而非仅依赖离散 token 监督。通过这种方式，弥合离散 token 与连续状态表示之间的语义鸿沟。
目标：在保留 MTP 促进世界模型构建优势的同时，消除结构性幻觉带来的负面影响。

📊 实验结果

  • 在合成图谱和真实曼哈顿出租车行程数据集上验证。
  • LSE-MTP 有效提升表示对齐质量，减少结构性幻觉，增强对扰动的鲁棒性。
