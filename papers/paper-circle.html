TL;DR 多智能体文献发现与分析系统：Discovery Pipeline（多源检索 + 多标准评分）+ Analysis Pipeline（结构化知识图谱 + 图感知 QA），全步骤输出 JSON/CSV/BibTeX/Markdown/HTML，完全可复现。ACL 2026 Oral。

📌 研究背景与动机

科学文献的快速增长使研究者难以高效发现、评估和综合相关工作。LLM 在理解用户意图和工具调用上展现出强大潜力，Paper Circle 旨在将这种能力系统化地应用于学术文献管理全流程。

⚙️ 系统架构

Discovery Pipeline：
  • 离线 + 在线多源检索（ArXiv、Semantic Scholar 等）
  • 多标准评分（相关性、新颖性、引用影响力等）
  • 多样性感知排序，避免结果同质化
  • 结构化输出（JSON/CSV/BibTeX/Markdown/HTML）
Analysis Pipeline：
  • 将单篇论文转化为带类型节点的知识图谱（概念、方法、实验、图表）
  • 图感知 QA：基于知识图谱结构进行问答
  • 覆盖度验证：检测综述是否遗漏重要相关工作
实现：基于 coder LLM 的多智能体编排框架，每个 agent 步骤均生成完全可复现的同步输出。

📊 实验结果

  • 在论文检索任务上报告 Hit Rate、MRR、Recall@K。
  • 在综述生成任务上评估覆盖质量。
  • 更强的 agent 模型带来一致性提升，验证框架可随模型进化持续改善。
