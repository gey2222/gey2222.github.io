研究方向

已有方法的局限性

  • 迭代方法慢：ASVD、SVD-LLM 等激活感知 SVD 方法需对每个权重矩阵迭代优化，压缩速度极慢（数小时至数天）。
  • 快速方法次优：随机化 SVD 等快速方法牺牲了理论最优性，在激活感知场景下精度损失较大，且缺乏理论保证。
  • 缺乏自适应 rank 分配：现有方法缺乏对各层「有效秩」的系统分析，往往采用统一的 rank 分配策略，无法针对不同层的可压缩性进行自适应优化。

本文核心问题

能否在激活感知的低秩 LLM 压缩中同时实现理论最优解（精度损失最小）和高速压缩，而非在两者之间取舍？

⚙️ 核心方法

整体框架

核心洞察：激活感知 SVD 的最优解可以用闭合形式表达——只需对输入激活的协方差矩阵做一次特征值分解即可得到理论最优低秩投影方向，无需任何迭代优化。将这个「一次性」操作与增量协方差聚合结合，就得到了既快又优的 Swift-SVD。

关键模块

① 增量激活协方差聚合：在校准数据集上逐批次更新各层输入激活的协方差矩阵 Σ_l = Σᵢ xᵢxᵢᵀ，无需存储所有激活，内存友好；聚合完成后矩阵不再变化。
② 闭合形式激活感知 SVD：对聚合的协方差矩阵进行单次特征值分解，得到激活感知的最优低秩基底。在信息论意义上等价于最小化激活加权重构误差，且只需执行一次 EVD 而非迭代优化。
③ Effective Rank 动态 Rank 分配：利用各层协方差矩阵的奇异值分布（熵归一化指标）量化该层的「有效秩」——即真正包含信息的维度数。可压缩性高的层分配低 rank，信息密集层保留高 rank，实现全局压缩比的最优分配。

关键公式

激活感知低秩压缩的最优解：
min_{Ŵ: rank(Ŵ)≤r} ‖W - Ŵ‖²_Σ，其中 ‖A‖²_Σ = tr(A Σ Aᵀ)

Swift-SVD 闭合形式解：对 Σ = VΛVᵀ，最优压缩为 Ŵ = W·V_r·V_rᵀ
其中 V_r 为协方差矩阵前 r 个特征向量（单次 EVD 得到）

与已有方法的本质区别

ASVD 等迭代方法通过反复优化逼近最优解；Swift-SVD 认识到「协方差矩阵可以预先聚合，EVD 只需执行一次」，将迭代问题转化为解析问题。速度提升 3-70x 的本质是：用一次矩阵分解替代了数十次迭代优化。
