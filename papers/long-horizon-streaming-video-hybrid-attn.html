Long-Horizon Streaming Video Generation via Hybrid Attention with Decoupled Distillation

Ruibin Li, Tao Yang, Fangzhou Ai, Tianhe Wu, Shilei Wen, Bingyue Peng, Lei Zhang · 2026-04

视频生成 · 长视频 · 流式 混合注意力 · 解耦蒸馏 · 时序一致性 arXiv 2504.07929 ↗

TL;DR

分钟级视频生成需要跨越数千帧的时序一致性，全注意力计算不可行。本文提出混合注意力（局部密集 + 全局稀疏）保持时序连贯，解耦蒸馏将教师全注意力知识迁移到学生稀疏注意力，内容与运动分支分别蒸馏。生成速度提升 2.8x，DOVER 主观质量高于基线 4.2 分。

研究背景

  • 长视频生成挑战：1 分钟 24fps 视频 = 1440 帧，DiT 全注意力复杂度 O(1440²) 完全不可行
  • 时序一致性需求：视频不同于图像批次，帧间运动和外观必须连贯，局部注意力会产生"闪烁"伪影
  • 流式生成需求：用户需要边生成边播放，不能等待完整视频生成完毕

核心方法

混合注意力（Hybrid Attention）：

对帧序列 F = {f_1, ..., f_T}：

局部窗口注意力（相邻 w 帧内）：
A_local(i) = softmax(Q_i K_{i-w:i+w}^T / √d) V_{i-w:i+w}

全局稀疏注意力（每 s 帧采一个锚帧）：
A_global(i) = softmax(Q_i K_anchor^T / √d) V_anchor
where K_anchor = {K_j | j mod s == 0}

混合输出：A = α·A_local + (1-α)·A_global

解耦蒸馏（Decoupled Distillation）：

  • 内容分支蒸馏：匹配教师（全注意力）和学生（混合注意力）的 per-frame 特征图
  • 运动分支蒸馏：匹配帧间光流一致性，防止混合注意力破坏运动轨迹
  • 两分支独立 loss 权重：避免内容蒸馏和运动蒸馏相互干扰

实验结果

  • 速度：相比全注意力基线，1 分钟视频生成速度 2.8x（单 A100，720p 24fps）
  • DOVER 主观质量：混合注意力+蒸馏 = 73.2 vs. 全注意力基线 69.0（+4.2 分）
  • Temporal SSIM：帧间一致性 0.87 vs. 仅局部注意力 0.79（+10%）
  • 文本对齐：CLIP 文本-帧相似度 0.312 vs. 基线 0.305

核心贡献

  1. 提出局部密集 + 全局稀疏混合注意力，兼顾效率与长程时序一致性
  2. 解耦蒸馏框架分离内容与运动知识迁移，首次在视频生成中使用
  3. 实现流式长视频生成，首帧延迟 < 5s，边生成边可播放
  4. 在 DOVER/EvalCrafter 等评估集上超越全注意力基线质量

局限性

  • 全局稀疏采样率 s 是固定超参，无法自适应场景（快动作需更密集采样）
  • 蒸馏需要全注意力教师模型，增加训练成本
  • 超过 2 分钟的视频生成时序一致性仍有退化趋势
