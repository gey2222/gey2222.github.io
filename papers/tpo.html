Target Policy Optimization (TPO)

目标策略优化
Jean Kaddour
2026-04-07
arxiv 2604.06159 ↗
GitHub ↗
TL;DR 将 RL 中「哪些 completion 应获得概率质量」与「参数如何更新」解耦。先构造目标分布 q，再用交叉熵拟合策略到 q。在稀疏奖励场景下显著优于 PPO/GRPO/DG，简洁且高效。

📌 研究背景与动机

在 RL 中，给定 prompt 后对模型采样一批 completion 并打分，需要回答两个问题：(1) 哪些 completion 应该获得更高概率？(2) 参数应该如何移动来实现这种改变？标准策略梯度方法将两个问题耦合，导致更新可能因学习率、clipping 等超参数过冲或欠冲，在稀疏奖励下尤为明显。

⚙️ 核心方法

目标分布构造：给定打过分的 completions，TPO 构造目标分布 qi ∝ piold exp(ui)，其中 ui 为归一化奖励信号。
策略拟合：用交叉熵将当前策略拟合到目标分布 q，而非直接优化奖励。损失对采样 completion logits 的梯度为 pθ − q，当策略匹配目标分布时自然归零，天然防止过拟合。

关键公式

目标分布：q_i ∝ p_i^old · exp(u_i)
梯度：∂L/∂logit_i = p^θ_i − q_i

📊 实验结果

  • 表格 bandit、Transformer 序列任务、十亿参数 LLM RLVR 上均有测试。
  • 简单任务上与 PG/PPO/GRPO/DG 持平。
  • 稀疏奖励场景下显著优于所有对比方法。
