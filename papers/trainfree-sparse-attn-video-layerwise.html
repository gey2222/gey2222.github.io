Training-Free Sparse Attention for Fast Video Generation via Offline Layer-Wise Sparsity Profiling and Online Bidirectional Co-Clustering

Jiayi Luo, Jiayu Chen, Jiankun Wang, Zhibo Chen, Jianxin Li et al. · 2026-03

视频生成 · 稀疏注意力 · DiT 层级稀疏率 · 双向协同聚类 · 训练无关 arXiv 2503.14211 ↗

TL;DR

视频 DiT 不同层的注意力稀疏特性差异极大，统一稀疏率导致高质量层被过度压缩。本文提出离线逐层最优稀疏率分析 + 在线双向协同聚类，在 Wan2.1 上实现 1.9x 推理加速，CLIP 相似度和时间一致性与全注意力无显著差距，无需任何训练。

研究背景

视频 DiT 稀疏注意力的关键洞察：

  • 层级差异：浅层注意力头倾向局部/纹理信息（高稀疏性），深层注意力头倾向全局语义（低稀疏性）
  • 现有方案问题：统一稀疏率（如全局 50%）会错误压缩深层关键注意力，导致语义失真
  • 双向聚类需求：单向聚类（只聚 Key 或只聚 Query）无法捕捉时空 token 的双边相似结构

核心方法

两阶段方案：

阶段一：离线层级稀疏率 Profile

  • 在代表性视频样本上运行全注意力，记录每层注意力分布特征（熵、集中度）
  • 用遗传算法搜索每层最优稀疏率，约束总体质量不低于阈值
  • Profile 结果存储为模型配置，推理时直接加载

阶段二：在线双向协同聚类

双向协同聚类（每层推理时执行）：
1. Q-聚类：K-Means(Q) → Q_labels（Q token 分组）
2. K-聚类：K-Means(K) → K_labels（K token 分组）
3. 协同矩阵：C[i,j] = mean_attn(Q_group_i, K_group_j)
4. 按层稀疏率 s_l，保留 C 中 top-(1-s_l) 的 (Q_group, K_group) 对
5. 只计算保留对内的精确注意力

时空结构感知：将帧内 token 和跨帧 token 分别聚类
保证帧内局部注意力和跨帧时序注意力分别优化

实验结果

  • 推理速度：Wan2.1（480p 81 帧）推理加速 1.9x，HunyuanVideo 加速 1.7x
  • 生成质量：VBench 总分与全注意力差距 <0.5%，CLIP 文本-视频相似度差距 <0.3%
  • 时间一致性：帧间一致性指标相比统一稀疏基线提升 3.8%
  • Profile 开销：离线 Profile 仅需 30 分钟（在 8 个 A100 上），一次 Profile 适用所有输入

核心贡献

  1. 首次系统分析视频 DiT 各层注意力稀疏特性的差异性
  2. 离线层级最优稀疏率搜索，一次 Profile 永久使用
  3. 时空感知双向协同聚类，保持帧内和帧间注意力质量
  4. 在多个主流视频生成模型上验证泛化性

局限性

  • 离线 Profile 针对特定视频内容分布，跨域泛化（如动画 vs 真实感）有待验证
  • 双向聚类增加在线开销，加速比 AdaCluster 等单向方法低
  • 不同分辨率/帧率需要重新 Profile，配置管理复杂
