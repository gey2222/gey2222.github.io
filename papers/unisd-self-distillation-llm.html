UniSD: Unified Self-Distillation Framework for LLMs

Yiqiao Jin et al. · 2026-05

自蒸馏 · 知识蒸馏 LLM 适配 · 无外部教师 arXiv 2605.06597 ↗

TL;DR

UniSD 是首个统一多种自蒸馏信号的 LLM 适配框架，无需任何外部教师模型。整合多教师一致性约束、EMA 教师稳定化、token 级对比学习、特征匹配和散度裁剪五大组件。跨 6 个 benchmark、6 个模型（3 个模型家族）验证，UniSD-full 比基础模型提升 +5.4 分，比最强单一基线提升 +2.8 分。

研究背景

  • 外部教师模型依赖：传统知识蒸馏需要更大的教师模型，但大型教师模型计算成本高且不易获取
  • 自蒸馏方法碎片化：现有自蒸馏方法（EMA 自蒸馏、对比学习等）各自独立设计，在不同场景下表现不稳定
  • 训练稳定性问题：自蒸馏中教师与学生使用同一网络，容易导致训练崩溃或知识退化
  • 多粒度蒸馏需求：token 级别和特征级别的知识对齐需要不同的损失函数，单一方法难以兼顾

核心方法

1. EMA 教师稳定化：

// 指数移动平均维护稳定教师 θ_teacher ← m · θ_teacher + (1-m) · θ_student // EMA 教师提供稳定的软标签，避免训练崩溃 soft_labels = Softmax(logits_teacher / T)

2. 多教师一致性约束：在训练过程中维护多个不同时间步的 EMA 快照作为多个教师，通过一致性损失约束学生输出与多教师预测的平均一致。

3. Token 级对比学习：

// 正样本：同一 token 在不同 dropout 视图下的表示 // 负样本：批次内其他 token 表示 L_contrast = -log(sim(z_i, z_i+) / ∑_j sim(z_i, z_j))

4. 特征匹配 + 散度裁剪：中间层特征对齐损失确保深层知识传递；散度裁剪（clip 大梯度）防止极端样本破坏优化稳定性。

实验结果

  • vs 基础模型（UniSD-full）：跨 6 个 benchmark 平均提升 +5.4 分
  • vs 最强单一基线：提升 +2.8 分，证明统一框架的协同增益
  • 跨模型家族泛化：在 3 个不同模型家族（6 个模型）上均取得一致提升
  • 消融实验：5 个组件各自均有贡献，完整组合效果最优
  • 训练稳定性：散度裁剪 + EMA 教师组合消除了训练过程中的性能崩溃现象

核心贡献

  1. 首个统一多种自蒸馏信号的完整框架，无需外部教师模型
  2. 系统分析了 EMA 稳定化、对比学习、特征匹配、散度裁剪的协同机制
  3. 跨 3 个模型家族 6 个模型的广泛验证，证明方法的通用性
  4. 为 LLM 的无外部依赖高效适配提供了完整的工程方案

局限性

  • 5 个组件同时激活带来额外的超参数调整负担（温度 T、动量 m、裁剪阈值等）
  • 多教师维护增加了训练时内存占用（多个 EMA 快照）
  • 在极小模型（<1B 参数）上自蒸馏的收益尚待验证
  • 对比学习需要较大的批次大小才能提供足够的负样本多样性
